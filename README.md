# miccast

A Raycast extension to quickly switch the active macOS microphone (input) and
speaker/headphone (output) device from a searchable list.

## Commands

- **Set Input Device** — switch the active microphone.
- **Set Output Device** — switch the active speakers/headphones.

## Development

```bash
npm install
npm run dev
```

Built on [`@raycast/api`](https://developers.raycast.com) and
[`macos-audio-devices`](https://github.com/karaggeorge/macos-audio-devices).
